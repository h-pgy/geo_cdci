# geo_cdci
O GeoCDCI automatiza a Certidão de Dados Cadastrais do Imóvel da SF/SUREM. O sistema realiza geocodificação híbrida e interpolação de logradouros, permitindo ao munícipe validar a localização exata do lote no mapa. A ferramenta aplica as regras da Ordem Interna 07/2018 para emitir certidões instantâneas ou direcionar casos complexos ao SP156.
